# GeometryApp - Система работы с геометрическими фигурами

## Описание
Система для создания и управления геометрическими фигурами с поддержкой файлового хранилища.

## Функциональность
- Создание прямоугольников, кругов, эллипсов
- Расчет площади и периметра
- Сохранение/загрузка в файл
- Сортировка и перемещение фигур

## Сборка и запуск
```bash
javac -d bin src/main/java/by/geometry/**/*.java
java -cp bin main.java.by.geometry.app.GeometryApp
